## Tech Stack

PM is built with:

- Go
- Echo
- Redis
- GORM
- Mockery
- Testify
- [Insomnia](https://insomnia.rest/)
- GitHub Actions
- Prometheus
- Grafana

```

```

## API Docs

## CI

PM uses Github Actions for running multiple workflows, like:
- Automated Testing
- Lint Checks
- Automatic API Documentation deployment (see [API Docs](#api-docs))
